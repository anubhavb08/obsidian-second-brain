## Current Working Model

This vault uses a simple LLM Wiki pattern:

- `10 Raw Sources` stores copied source snapshots.
- `20 Wiki` stores cleaned, linked knowledge pages.
- `00 Control` stores navigation, rules, and logs.
- `90 Templates` stores reusable note shapes.

Raw source snapshots are evidence. Wiki pages are the working memory layer.
