# Life Automation OS

This is the dashboard for automating day-to-day life.

## Current Status

Use this page to organize routines, reminders, file cleanup, personal operations, recurring tasks, and decision support.

## Suggested Operating Areas

- Daily and weekly review.
- File organization and cleanup.
- Reminders and recurring automations.
- Personal task capture.
- Personal knowledge intake.
- Health, learning, money, travel, and admin systems.

## Starter Workflow

1. Capture raw tasks and ideas in an inbox note.
2. Convert recurring items into routines or automations.
3. Keep one weekly review page for deciding what matters now.
4. Move stale items into archive instead of letting the dashboard become noisy.
