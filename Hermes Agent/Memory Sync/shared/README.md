# Hermes Shared Core Memory

This folder is the human-editable shared-memory control plane.

- `USER.md`: compact user-profile entries that should be copied into each Hermes profile where capacity allows.
- `MEMORY.md`: compact environment/project entries that should be copied into each Hermes profile where capacity allows.

Use Hermes native memory for profile-specific facts/personas. Use this shared core only for facts every agent should know.
Entries should be separated with a line containing only `§`.
